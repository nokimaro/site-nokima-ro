apiProductsCatalogFiltersGet

URL

https://market-mini-back-staging.stage.rccore.net/api/products/catalog/{filters}

HTTP Method

GET

Response Type

Parameters

  • Path: filters String

    список `slug` категорий и брендов разделённых `/`

  • Query: page Integer

    номер страницы, начиная с \"1\"

  • Query: sort_type String

    Тип сортировки: * `position` - По популярности. Больше position - популярнее * `price` - По цене

  • Query: sort_direction String

    Порядок сортировки: * `asc` - Ascending, from A to Z * `desc` - Descending, from Z to A
